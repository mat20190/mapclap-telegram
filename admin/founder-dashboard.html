Infrastructure

Код, база и релизы

Для настоящего ресурса нужно подключить GitHub для кода, Supabase/Firebase/Postgres для пользователей, S3/Cloudinary для фото, отдельный домен для админки и закрытую авторизацию основателя.

Firebase real-time база

Для живых данных создай Firebase-проект, включи Authentication, Firestore или Realtime Database, Storage для фото и вставь web config в файл admin/firebase-config.js.

Firebase: локальный режим, конфиг еще не подключен.
